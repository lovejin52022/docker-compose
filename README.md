# docker-compose
常用的开发工具集
- ES 
    > 主要集成了elasticsearch 以及 ik 分词工具环境 kibana ui 显示页面
- fastdfs
    > 主要集成分布式文件系统fastdfs环境
- jenkins
    > 主要集成开源自动化服务，构建、部署、自动化环境。
- jre
    > 运行java程序的环境 java runtime 环境。
- mongo
    > mongo数据库集成
- mysql
    > mysql数据库集成
- postgres
    > postgres数据库集成
- rabbitmq
    > 集成rabbitmq环境，主要和java程序组合使用。
- redis
    > 集成redis环境
- redis_batch
    > 集成redis集群环境demo，加入了哨兵。
- scrapyd
    > python爬虫框架
- sql_server
    > 微软的sqlserver的数据库环境
- zookeeper
    > 分布式协调服务环境，主要配合java的Dubbo框架组合使用,以及Kafka等框架。
- zookeeper_batch
    > 集成了高可用的集群环境，1 + 2n部署
